## Projects

### 🔐 Cipher
- **Path**: [`cipher/`](cipher/)
- A simple Caesar-style cipher that can **encrypt** and **decrypt** text using a numeric key.  
- Example usage:
```bash
./cipher -e "hello" 3   # Encrypt
./cipher -d "khoor" 3   # Decrypt
```
